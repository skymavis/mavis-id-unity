# SDK Playground

Ronin Waypoint SDK example using Unity for Android, iOS and desktop games.

## Prerequisites

- `Unity` : >= `2020.3.48f1`
- `iOS` : >= `13.0`
- `Android` : >= `Android API level 24`
- `Desktop` : >= `.Net 3.1`

## Start

In Unity Editor, go to **Build Settings** > choose the platform to deploy to > **Build and Run**.
